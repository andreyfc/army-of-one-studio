Política de Privacidade

Última atualização: 9 de abril de 2026

Este aplicativo foi desenvolvido com o objetivo de proporcionar uma experiência simples e segura para todos os usuários.

Coleta de dados

Este aplicativo não coleta, armazena ou compartilha dados pessoais dos usuários.

Dados do dispositivo

Nenhuma informação do dispositivo, como localização, contatos ou identificadores, é coletada ou armazenada.

Publicidade

Este aplicativo não exibe anúncios e não utiliza serviços de terceiros para fins publicitários.

Compras

O aplicativo é pago no momento da aquisição e não realiza coletas adicionais de dados para fins de cobrança fora das plataformas oficiais.

Crianças

Este aplicativo é adequado para todas as idades, incluindo crianças menores de 13 anos. Como não há coleta de dados, não há risco relacionado à privacidade infantil.

Segurança

Como nenhum dado é coletado, não há armazenamento de informações pessoais.

Alterações nesta política

Esta política pode ser atualizada ocasionalmente. Recomendamos que os usuários revisem esta página periodicamente.
